# Axios 学习复盘（面向“看懂项目代码”）

## 优点

1. 覆盖了项目高频结构：实例、拦截器、service 分层、错误处理。
2. 示例短小，便于你快速建立“阅读模式”。
3. 每层都对接了真实项目代码习惯。

## 当前不足

1. 还没有结合你正在做的具体业务项目做定制接口实战。
2. token 刷新示例是教学版，未覆盖并发刷新的完整队列方案。

## 下一步改进

1. 选你正在读的一个项目，我帮你逐文件映射到本教学包概念。
2. 补一份“你项目里的 axios 代码阅读清单”。

## 知识点分类

### 必须背下来

1. `params` 和 `data` 区别。
2. `axios.create` 的作用。
3. `error.response / error.request / error.message` 区别。

### 熟悉掌握

1. 请求/响应拦截器的常见职责。
2. service 层封装方式。
3. 并发请求和串行请求的判断与写法。

### 了解即可

1. token 自动刷新完整队列实现。
2. 请求取消在复杂页面交互中的细节。
3. 更高级的缓存与重试中间件。
